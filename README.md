# 🚀 AI Chatbot (Node.js + React)

This is a full-stack AI chatbot built using **React (frontend)** and **Node.js + Express (backend)**.  
It allows users to ask questions and get intelligent responses using the OpenRouter API.

🔗 **Live Demo:**  
[Click Here](https://chatbotfrontend-sn6p.onrender.com/)

---

## ✨ Features

- 💬 Real-time chatbot interaction  
- 🧠 AI-powered responses (via OpenRouter)  
- ⚡ Fast and responsive UI  
- 🔄 Backend API integration  
- 🌐 Fully deployed using Render  

---

## 🛠️ Tech Stack

- **Frontend:** React.js  
- **Backend:** Node.js, Express  
- **Database:** MongoDB  
- **AI API:** OpenRouter  
- **Deployment:** Render  

---

## 📌 Current Limitations

- Only supports **text input**
- No conversation memory yet
- Basic UI (no streaming responses)

---

## 🚧 Future Improvements

- Add chat history (memory)
- Improve UI/UX (typing animation, chat bubbles)
- Support file/image input
- Optimize API usage and performance
